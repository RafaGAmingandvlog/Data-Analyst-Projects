# Data-Analyst-Projects
📖 Overview
A comprehensive student grade management system built with:
Python (OOP)
SQLite database
Pandas for analysis
Matplotlib for visualization

🚀 Features
CRUD operations
Statistical analysis
Grade classification system
Data visualization dashboard
Export to CSV, Excel, JSON

📊 Analytical Capabilities
Mean, median, standard deviation
Quartile analysis
Class-based group analysis (SQL GROUP BY)
Ranking system
Trend visualization

🛠 Tech Stack
Python 3.x
SQLite
Pandas
NumPy
Matplotlib

▶️ How to Run
pip install -r requirements.txt
python src/Student's-Grade-Management-System.py
